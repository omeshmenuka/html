1. Introduction

Traditional doorbells require physical contact. This project creates a touchless doorbell using an ultrasonic sensor to detect a hand nearby and activate a buzzer. The system is compact, low-cost, and easy to build in a lab.

2. Problem Statement

  • Push-type doorbells require touching a common surface.
  • Touch surfaces can spread germs and bacteria.
  • A contactless solution improves hygiene and convenience.

3. Objectives

  • Detect a hand/object within a set distance using HC-SR04.
  • Trigger a buzzer as a doorbell alert.
  • Keep the design small and suitable for lab demonstration.

4. Components Required

  • Arduino Uno
  • Ultrasonic Sensor (HC-SR04)
  • Active Buzzer (recommended) or Passive Buzzer
  • Breadboard
  • Jumper wires
Optional: LED + 220Œ© resistor (visual ring indicator).

5. Working Principle

The HC-SR04 sends ultrasonic waves and listens for the echo. Arduino measures echo time and converts it to distance. If distance is less than a threshold (example: 10 cm), the buzzer turns ON.

  • Send trigger pulse
  • Read echo duration
  • Calculate distance
  • Compare with threshold
  • Ring buzzer (doorbell)
